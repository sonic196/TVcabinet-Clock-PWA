# TVcabinet Clock - iOS 風格床頭時鐘 PWA

一個現代化、功能豐富的漸進式網頁應用（PWA）時鐘，專為床頭櫃使用而設計，靈感來自 iOS 時鐘應用。

## ✨ 主要功能

### 時間顯示
- **12/24小時制切換**：支援 12 小時制（AM/PM 顯示於日期前方）和 24 小時制
- **智慧模式切換**：縱向模式自動切換為 24 小時制，橫向模式可自由選擇
- **優雅的數字過渡動畫**：數字切換時帶有隨機旋轉和位移的交叉淡入淡出效果
- **智慧閃爍動畫**：
  - **縱向模式**：日期閃爍
  - **橫向模式**：冒號閃爍
  - **柔和效果**：透明度在 100% 與 10% 之間變化，視覺更舒適

### 視覺效果
- **20 組精美配色主題**：
  - Classic Blue、Sunset Vibes、Aurora Green、Royal Purple
  - Electric Violet、Golden Sand、Midnight Neon、Sweet Morning
  - Spring、Coral Sunset、Ocean Breeze、Tropical Fire
  - Deep Violet、Mint Fresh、Candy Dream、Sky Blue
  - Hot Red、Emerald、Neon Purple、Golden Orange
- **雙色調設計**：
  - **縱向模式**：小時使用顏色 A，分鐘使用顏色 B
  - **橫向模式**：第 1/3 位數字使用顏色 A，第 2/4 位數字使用顏色 B
- **自動換色**：每小時自動切換到下一組配色
- **夜間模式**：深紅色低亮度顯示，適合夜間使用，保護眼睛

### 顯示優化
- **縱向模式 (Portrait) 極致優化**：
  - **超大字體**：75vw 巨型數字
  - **緊湊交疊**：-8vw 負邊距，製造強烈的視覺衝擊
  - **垂直佈局**：小時在上，日期在中，分鐘在下
  - **強制 24H**：縱向模式自動切換為 24 小時制
  - **單排按鈕**：底部橫向排列，節省空間
- **橫向模式 (Landscape) 經典佈局**：
  - **安全區域**：適配 iPhone 動態島與瀏海
  - **日期位置**：右下角顯示
  - **AM/PM 整合**：12H 制時，AM/PM 顯示於日期前方（例：`PM｜12月2日 週二`）
- **防烙印保護**：每分鐘隨機微移時鐘位置（±20px），防止 OLED 螢幕烙印
- **PWA 支援**：可安裝到主畫面，離線使用，全螢幕模式

## 🎨 設計理念

### 視覺美學
- **浮動效果**：使用 `mix-blend-mode: screen` 實現數字發光混色效果
- **柔和過渡**：所有動畫皆使用緩動函數，確保視覺流暢
- **極簡介面**：主畫面僅顯示時間和日期，控制按鈕隱藏，點擊螢幕才顯示

### 技術選型
- **純 HTML/CSS/JavaScript**：無框架依賴，輕量快速
- **Google Font Nunito**：模仿 SF Pro Rounded 的圓潤字體
- **CSS 變數 + color-mix()**：動態配色系統，現代瀏覽器原生支援
- **Service Worker**：實現離線快取功能

### 用戶體驗
- **一鍵切換**：所有功能均可通過單一按鈕快速開關
- **視覺反饋**：按鈕狀態與功能狀態即時同步
- **跨平台相容**：響應式設計，支援手機、平板、電腦

## 📖 使用說明

### 基本操作
1. **開啟時鐘**：用瀏覽器打開 `index.html`
2. **顯示控制按鈕**：點擊螢幕任意位置
3. **切換功能**：
   - 🎨 **換色**：循環切換 20 組配色
   - 🕒 **24H/12H**：切換時間格式（縱向模式強制 24H）
   - 📅 **日期**：開啟/關閉日期顯示
   - ✨ **閃爍**：開啟/關閉閃爍效果（縱向閃日期，橫向閃冒號）
   - 🌙 **夜間模式**：切換為深紅色低亮度顯示

### 推薦設置
- **床頭使用**：開啟夜間模式 + 12 小時制
- **桌面時鐘**：開啟日期 + 24 小時制 + 選擇喜歡的配色
- **極簡模式**：隱藏日期 + 關閉閃爍 + 單一配色

## 🚀 安裝與部署

### 方法一：本地使用
1. 下載所有檔案到同一資料夾：
   ```
   index.html
   manifest.json
   sw.js
   icon.png
   ```
2. 用瀏覽器開啟 `index.html`

### 方法二：安裝為 PWA（推薦）

#### iOS (Safari)
1. 用 Safari 開啟時鐘網頁
2. 點擊下方分享按鈕 📤
3. 選擇「加入主畫面」
4. 從主畫面啟動，享受全螢幕體驗

#### Android (Chrome)
1. 用 Chrome 開啟時鐘網頁
2. 點擊右上角選單 ⋮
3. 選擇「安裝應用程式」或「加到主畫面」
4. 從主畫面啟動

### 方法三：部署到網路伺服器

#### GitHub Pages（免費）
1. 建立 GitHub Repository
2. 上傳所有檔案到 Repository
3. 到 Settings → Pages
4. 選擇 `main` 分支，儲存
5. 等待幾分鐘，從提供的網址訪問
