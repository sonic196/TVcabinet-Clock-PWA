# TVcabinet Clock - iOS 風格床頭時鐘 PWA

一個現代化、功能豐富的漸進式網頁應用（PWA）時鐘，專為床頭櫃使用而設計，靈感來自 iOS 時鐘應用。

## ✨ 主要功能

### 時間顯示
- **12/24小時制切換**：支援 12 小時制（含 AM/PM 指示器）和 24 小時制
- **優雅的數字過渡動畫**：數字切換時帶有隨機旋轉和位移的交叉淡入淡出效果
- **冒號閃爍動畫**：可開關的秒數閃爍效果（2 秒週期，完全透明到不透明）

### 視覺效果
- **20 組精美配色主題**：
  - Classic Blue、Sunset Vibes、Aurora Green、Royal Purple
  - Electric Violet、Golden Sand、Midnight Neon、Sweet Morning
  - Spring、Coral Sunset、Ocean Breeze、Tropical Fire
  - Deep Violet、Mint Fresh、Candy Dream、Sky Blue
  - Hot Red、Emerald、Neon Purple、Golden Orange
- **雙色調設計**：第 1/3 位數字使用顏色 A，第 2/4 位數字使用顏色 B，冒號與 AM/PM 使用混合色（80% 白 + 20% 顏色 A）
- **自動換色**：每小時自動切換到下一組配色
- **夜間模式**：深紅色低亮度顯示，適合夜間使用，保護眼睛

### 便利功能
- **日期顯示開關**：可隱藏日期，時鐘自動放大填滿螢幕
- **螢幕常亮**：防止螢幕自動休眠
- **防烙印保護**：每分鐘隨機微移時鐘位置（±20px），防止 OLED 螢幕烙印
- **PWA 支援**：可安裝到主畫面，離線使用，全螢幕模式

### 排版優化
- **12 小時制專屬排版**：
  - 字體放大至 38.5vw（比 24H 模式大 10%）
  - 冒號間距放寬至 2vw
  - AM/PM 指示器距離時間 4vw
- **24 小時制緊湊排版**：標準 35vw 字體，緊密排列
- **日期隱藏時自動放大**：隱藏日期後，時鐘放大至 45vw（12H 模式為 48vw）

## 🎨 設計理念

### 視覺美學
- **浮動效果**：使用 `mix-blend-mode: screen` 實現數字發光混色效果
- **柔和過渡**：所有動畫皆使用緩動函數，確保視覺流暢
- **極簡介面**：主畫面僅顯示時間和日期，控制按鈕隱藏，點擊螢幕才顯示

### 技術選型
- **純 HTML/CSS/JavaScript**：無框架依賴，輕量快速
- **Google Font Nunito**：模仿 SF Pro Rounded 的圓潤字體
- **CSS 變數 + color-mix()**：動態配色系統，現代瀏覽器原生支援
- **Service Worker**：實現離線快取功能
- **NoSleep 視頻技巧**：使用隱藏影片保持螢幕常亮

### 用戶體驗
- **一鍵切換**：所有功能均可通過單一按鈕快速開關
- **視覺反饋**：按鈕狀態與功能狀態即時同步
- **跨平台相容**：響應式設計，支援手機、平板、電腦

## 📖 使用說明

### 基本操作
1. **開啟時鐘**：用瀏覽器打開 `index.html`
2. **顯示控制按鈕**：點擊螢幕任意位置
3. **切換功能**：
   - 🎨 **換色**：循環切換 20 組配色
   - 🕒 **24H/12H**：切換時間格式
   - 📅 **日期**：開啟/關閉日期顯示
   - ✨ **閃爍**：開啟/關閉冒號閃爍
   - 🌙 **夜間模式**：切換為深紅色低亮度顯示
   - ⚡ **保持常亮**：啟用後螢幕不會自動休眠

### 推薦設置
- **床頭使用**：開啟夜間模式 + 關閉閃爍 + 12 小時制
- **桌面時鐘**：開啟日期 + 24 小時制 + 選擇喜歡的配色
- **極簡模式**：隱藏日期 + 關閉閃爍 + 單一配色

## 🚀 安裝與部署

### 方法一：本地使用
1. 下載所有檔案到同一資料夾：
   ```
   index.html
   manifest.json
   sw.js
   icon.png
   ```
2. 用瀏覽器開啟 `index.html`

### 方法二：安裝為 PWA（推薦）

#### iOS (Safari)
1. 用 Safari 開啟時鐘網頁
2. 點擊下方分享按鈕 📤
3. 選擇「加入主畫面」
4. 從主畫面啟動，享受全螢幕體驗

#### Android (Chrome)
1. 用 Chrome 開啟時鐘網頁
2. 點擊右上角選單 ⋮
3. 選擇「安裝應用程式」或「加到主畫面」
4. 從主畫面啟動

#### 桌面版 (Chrome/Edge)
1. 開啟時鐘網頁
2. 網址列右側會出現安裝圖示 ＋
3. 點擊安裝
4. 在應用程式列表中啟動

### 方法三：部署到網路伺服器

#### GitHub Pages（免費）
1. 建立 GitHub Repository
2. 上傳所有檔案到 Repository
3. 到 Settings → Pages
4. 選擇 `main` 分支，儲存
5. 等待幾分鐘，從提供的網址訪問

#### Netlify/Vercel（免費）
1. 拖曳資料夾到 Netlify Drop 或 Vercel
2. 自動部署完成
3. 使用提供的網址訪問

#### 自有伺服器
1. 將所有檔案上傳到網頁伺服器
2. 確保伺服器支援 HTTPS（PWA 必需）
3. 訪問網址即可使用
